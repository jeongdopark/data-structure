# Linked List
각 노드가 데이터와 다른 노드를 가리키는 포인터를 가지고 있는 선형자료구조이다.
포인터로 이어진 특징으로 동적으로 메모리 할당이 가능하며, 메모리 이동 없이 삽입과 삭제가 가능하다.
